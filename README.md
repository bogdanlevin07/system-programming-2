NamedPipe Messenger

Простое консольное приложение на C# для обмена сообщениями между клиентом и сервером через именованные каналы (Named Pipes). Поддерживает три команды: получение времени, эхо-ответ и завершение сеанса.

Описание

- Сервер ожидает подключения клиента по каналу `mypipe`, обрабатывает команды и отправляет ответы.

- Клиент подключается к серверу, отправляет команды и выводит полученные ответы.

- Обмен данными происходит в текстовом режиме (построчно).

Требования

- [.NET Framework](https://dotnet.microsoft.com/download) (4.5+) или [.NET Core / .NET 5+](https://dotnet.microsoft.com/download) – подойдёт любая версия, поддерживающая `System.IO.Pipes`.

- Компилятор C# (`csc` из Visual Studio Build Tools или `dotnet` SDK).
